First start

Remember/copy your tehybug key: config_system.bug_key

Step 1.
Provide credentials of your home wifi network at the Configure WiFi page.

Step 2.
Check whether the TeHybug was able to connect to your WiFi network (it supports only 2.4G WiFi) Node info page. There should be an TeHyBugs IP adress on you network which you can use to continue the configuration.

Step 3.
Select the sensor that you use at Sensors configuration page.

Step 5.
Check whether the sensor is serving the data properly at Current environment data page.

Step 6.
Select the desired data serving mode, like the "Default configuration" for tehybug.com cloud server or other custom data serving services. If you dont have an own server or mqtt broker, select the Default configuration. Data frequency setting means, how often the sensor data should be served. The higher you set the data frequency the longer the battery will last.

Step 7 and 8.
Activate live mode and restart your TeHyBug by clicking the restart link or hitting the reset button. Your TeHyBug will reboot and start serving the sensor data.

More infos about TeHyBug can be found at tehybug.com or at the tindie store: https://www.tindie.com/stores/gumslone/
